# 🏍️ Suzuki Inventory Manager

Este projeto é uma ferramenta de automação para gestão de ativos, desenvolvida para auxiliar no controle de estoque da concessionária **Suzuki de São João da Boa Vista**.

### 🚀 Funcionalidades
- Cadastro de modelos e cores de motocicletas.
- Listagem em tempo real dos veículos em pátio.
- Cálculo automático do valor total investido no estoque (análise financeira rápida).

### 🛠️ Tecnologias
- **Linguagem:** Python 3
- **Conceitos:** Lógica de programação, Estrutura de dados (Listas e Dicionários).
